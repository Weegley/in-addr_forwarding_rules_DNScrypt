# in-addr_forwarding_rules_DNScrypt
Generate forwarding rules for reverse lookups for dnscrypt-proxy 2 from ASN prefixes
## Prequisites:
1. bash to be used
2. [jq 1.6+](https://stedolan.github.io/jq/)
3. [curl](https://curl.se/)
